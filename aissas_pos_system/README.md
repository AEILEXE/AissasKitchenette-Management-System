## Quick start (dev mode)

From inside this directory (`aissas_pos_system/`) with the venv active:

```bash
python main.py
```

## Build the EXE + installer

```bash
build.bat
```

Produces:
- `dist/AissasKitchenette.exe` — standalone Windows EXE
- `dist/AissasKitchenette_Setup.exe` — Windows installer (requires Inno Setup 6)
